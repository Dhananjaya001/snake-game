A modern take on the classic Snake game built with Python and Pygame! Control the snake, eat food to grow longer, and try to beat your high score without colliding with walls or yourself.

Features

Smooth Gameplay: Responsive controls for an enjoyable gaming experience.
Dynamic Graphics: Minimalistic and visually appealing design with vibrant colors.
Customizable Speed: Adjust the game speed to match your preferred challenge level.
Replay Option: Easily restart the game after losing.
Requirements

Python 3.x
Pygame library (pip install pygame)
How to Play

Use the arrow keys to control the snake's direction.
Eat the yellow food blocks to grow and increase your score.
Avoid running into the walls or your own tail!
Getting Started

Clone this repository and run the snake_game.py file:

git clone https://github.com/your-username/snake-game.git
cd snake-game
